# Todo List App

할 일을 추가하고 완료 상태를 관리할 수 있는 간단한 투두리스트 앱입니다.

## 🔗 Demo

https://unibure-todo.vercel.app

## ✨ Features

- 할 일 추가 / 삭제
- 체크박스를 통한 완료 상태 관리
- 전체 삭제 기능 + 커스텀 모달 UI 적용
- localStorage를 활용한 데이터 저장

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage
- Vercel

## 📌 Key Points

- 이벤트 위임 방식으로 동적 리스트 요소 처리
- 모달 컴포넌트 구현을 통해 UX 개선
- 새로고침 후에도 데이터 유지되도록 설계
